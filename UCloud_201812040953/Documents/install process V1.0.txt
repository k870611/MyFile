1.将formal 资料夹放于桌面

2.安装：(资料夹位于：%UserProfile%\Desktop\Formal\KyLin\install：)

	b.python-3.4.4.amd64.msi
					设定安装路径为：预设路径 - C:\Python34
					设定环境变数: 控制面板\所有控制面板项\系统\高级系统设置\高级\环境变量 在path 加入 C:\Python34 & C:\Python34\Scripts
					
	c.ChromeStandaloneSetup.exe
	d.python_lib(DB, Flask, socketio) by pip

	e.Java：jre-8u172-windows-x64.exe

3.设定开机启动项：(regedit >HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Run)	
	a.新建字符串值
		(1)名称：beerservice，值：%UserProfile%\Desktop\Formal\KyLin\beerservice\beer_service.cmd
		(2)名称：run_auto，值 :%UserProfile%\Desktop\Formal\KyLin\app\run_auto.bat

4.设定mysql event scheduler
	a.设定my.ini(C:\ProgramData\MySQL\MySQL Server 5.7\my.ini)
	b.搜寻[mysqld]
	c.在[mysqld]下方加入event_scheduler=on

5.设定settings.cmd(%UserProfile%\Desktop\Formal\KyLin\beerservice\settings.cmd)
	SET MYSQL=C:\Program Files\MySQL\MySQL Workbench 6.3 CE\mysql.exe(若安装路径有更动请调整路径)
	SET HOST=localhost(若使用本机则无须更改)
	SET PORT=3306 (若mysql端口有换请调整)
	SET DBUSER=root (若要更换使用者请将DBUSER 改为您要使用的account & password)
	SET DBPWD=123456 (系统安装时设定为123456)
	
	SET PYTHON=C:\Python34\python.exe (若安装路径有更动请调整路径)
	SET IPMITOOL=..\tools\ipmitool.exe (若有移动tools资料夹必须调整路径)
	
6.设定dbSettingsInfo.py (%UserProfile%\Desktop\Formal\KyLin\app\dbSettingsInfo.py)
	DATABASE = {
		'drivername': 'mysql+mysqlconnector',# 若使用不同mysql connector 需更换成所使用的驱动
		'host': 'localhost', # host, port, username, password 需遵循settings.cmd 中的HOST,PORT, DBUSER, DBPWD
		'port': '3306',
		'username': 'root',
		'password': '123456',
		'database': 'server_management'
	}
	
7.执行auto-build-db.py
	a.open cmd
	b.python %UserProfile%\Desktop\Formal\KyLin\beerservice\auto-build-db.py
